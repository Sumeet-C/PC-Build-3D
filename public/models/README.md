# 3D Models Directory

This directory is reserved for future realistic GLB/GLTF 3D assets.

Currently, the application uses **procedural geometry** (transparent geometric shapes) for all PC components. When you add `.glb` files here, update the `model_path` field in the component seed data to reference them.

## Expected Files
- `cpu.glb`
- `gpu.glb`
- `ram.glb`
- `motherboard.glb`
- `storage.glb`
- `psu.glb`
- `case.glb`
- `cooling.glb`
